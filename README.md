# Toronto BikeShare Streamlit Web App (Tutorial-Based Learning Project)

An interactive Streamlit web application built by following Maggie In Data's tutorial using Toronto BikeShare GTFS data.

This project helped me gain hands-on experience with Streamlit, data preprocessing, interactive maps, KPI metrics, user-centric design, and handling edge-case logic.

Although the project idea and overall structure were tutorial-guided, I implemented the application myself and understood each component in detail.

Key learnings:
• Building interactive web apps with Streamlit
• Working with real-world GTFS data
• Data manipulation with Pandas
• Creating KPI dashboards
• Interactive maps with Folium
• Designing user-friendly interfaces
• Implementing filtering and edge-case handling

Tutorial credit:
Maggie In Data – "Code Your First Streamlit Web App with Python"
